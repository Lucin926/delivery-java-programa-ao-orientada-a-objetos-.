# Sistema de Delivery (POO)

Projeto simples em Java utilizando:

- Herança
- Polimorfismo
- Sobrescrita de métodos

## 📌 Funcionalidades

- Pedido com entrega normal
- Pedido com entrega expressa
- Pedido com retirada no local

Cada tipo calcula o valor total de forma diferente.

## 🚀 Tecnologias

- Java
